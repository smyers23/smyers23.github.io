Help me please...

Okay, so you ran into some problems, no worries. This page is split into two sections. The first is for help with getting the game running in the first place, the second is for help with the game itself.

Get the game running

If you clicked on the "Enter the Hunt" button on the last page and it didn't work or the game didn't load there are a few things you can try. (There's actually a special bonus easter egg if you use options 1 or 2a.)

  1. Download the EXE and play it on your computer. Note this will only work on Windows.
  2. Download the game file and use a different interpreter. You have a few options here.
    • Download QTADS. Then just open QTADS, click on Game, click on Open, and select the CrabHunt.t3 file you downloaded above.
    • Open Parchment, click on Play File on Your Device, and select the CrabHunt.t3 file you downloaded above.
    • Open ElseIFPlayer, click on Local File, and select the CrabHunt.t3 file you downloaded above. Note the gameplay will be a little funky if you use this method.

Because the game is hard

Playing text adventure games can be a little unintuitive. If you're stuck there's a few things you can do.

  1. First, make sure to type ABOUT and INSTRUCTIONS in the game and read everything. This will help you get a sense of how to approach the game.
  2. If you're struggling with the puzzles first type HINT in the game. This will open a menu with different topics. If you select a topic it will show a hint on how to proceed on that topic. You can get progressively more instructive hints by clicking multiple times.
  3. Try looking at the map of the game world. This does contain spoilers as every important game item is marked on the map.
  4. Read the complete walkthrough here.
