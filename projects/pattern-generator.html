FlossCross Pattern Generator

Originally Posted: July 12, 2026

Ever since I started cross stitching, I've wanted an easy way to turn pixel arts images I find online into easy-to-follow cross stitch patterns. After a long time of trying to find the perfect program, I gave up and decided to make my own. FlossCross Pattern Generator is a Python script I wrote that will turn a pixel art image (or any image for that matter) into a cross stitch pattern that can be used with the free website FlossCross. (Note that this project is not in anyway affiliated with FlossCross or its creators.) Anyway, its a command-line program without a GUID, and it requires you have Python installed to run, but if you're interested in checking it out you can do so here!
